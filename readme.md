# McClane
![John McClane](https://theblemish.com/images/2017/09/die-hard-bruce-willis.jpg)

 Dockerfile for image intended to be used by [CircleCI](https://circleci.com) build and deploy jobs.

 Runs on Debian.
 
 Supports the following tools:
  - nodejs
  - npm
  - phantomjs
  - clojure
  - leiningen
  - terraform
  - awscli
  - awsebcli
  - Ruby
  - Bundler
  - chrome-headless
  - chromedriver
  - firebase-tools
  - docker
  - maven
  - Karma
  - shadow-cljs
  - Google Cloud SDK
  - vim
  - PostgreSQL
